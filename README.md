



# 一、项目说明

本仓库用于统一管理 **资金大模型 / 资金流智能体产品原型**，集中维护原型访问入口，方便业务人员、产品团队及技术团队进行 **需求评审、产品讨论与功能验证**。

所有原型均基于 **Google AI Studio** 构建。

# 三、原型模块导航

当前系统原型主要包含以下业务模块：

| 模块      | 功能说明                      | 访问入口   |
| ------- | ------------------------- | ------ |
| 资金穿透看板--费控业务线   | 费用报销与资金流数据关联展示            | 进入系统查看 |
| 资金穿透看板--合同库     | 合同信息与资金流水关联展示             | 进入系统查看 |

当前原型为 **统一系统原型入口**，进入系统后可查看各业务模块页面。

# 四、系统功能结构

资金大模型原型主要围绕 **资金流数据穿透 + 业务系统关联 + 智能分析能力**构建：

```
资金数据
   │
   ├── 银行流水
   ├── 费控系统
   ├── 合同系统
   ├── OA系统
   └── 客户信息

资金大模型能力
   │
   ├── 资金流识别
   ├── 资金用途分类
   ├── 资金关联穿透
   └── 资金风险识别

资金智能体应用
   │
   ├── 资金穿透看板
   ├── 投资分析智能体
   ├── 业务流程关联
   └── 资金流数据分析
```
